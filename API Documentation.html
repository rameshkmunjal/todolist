API Documentation

Base URL : /todolist

1. SIGN-UP (/sign-up)

  • POST

Params

param Mandatory Type Description
email Yes string. email to generate OTP for
password Yes string Password
firstName Yes string First Name
lastName Yes string Last Name
Country Yes string eg: India
countryCode Yes string eg: 91
mobile Yes string eg 9999999999

Response:

200:
{“message”: “User Created successfully”, “userId”: “random short id”}
400:
User with this email already exists
400
Email address is not as per format
400
Password is not as per format

2. login (/login)

  • POST

Params

param Mandatory Type Description
email Yes string. email to generate OTP for
password Yes string Password

Response:

200 :
Login successful
400 :
Wrong password. Login failed
500:
Some error occurred
500
Failed to generate Token

3. logout (/logout)

  • POST

Params

param Mandatory Type Description
userId Yes string Identifies a user

Response:

200:
User logged out successfully

4. MatchOTP (/matchOTP)

  • POST

Params

param Mandatory Type Description
userId Yes string. Identifies user
otp Yes numeric Uniquely generated for a user

Response:

200:
OTP matched

403:
OTP does not match

5. ResetPassword (/resetPassword)

  • POST

Params

param Mandatory Type Description
email Yes string. email to reset password for
code Yes string Security Code Uniquely sent over an email

Response:

200 :
Password changed successfully

500 :
Some error occurred

6. demandOTP (/demandOTP)

  • GET

Params

param Mandatory Type Description
email Yes string. email to generate OTP for

Response:

200:
OTP sent to email address

500:
Create OTP - Some error occurred

7. getAllUsers (/getAllUsers)

Generates list of friends for a userId

  • GET

Params

param Mandatory Type Description
userId Yes string. userId to generate list of friends for

Response:

200:
{
“error”:false,
“status”:200,
“message”:“User’s friend data fetched successfully”
“friends” : [{“friendId” : “”, “friendName”:""},
{“friendId” : “”, “friendName”: “”}]
}

400:
No user data found
